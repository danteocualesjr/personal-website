# Welcome to My Blog

This is your first blog post! You can write in Markdown and use React components.

## Features

- **Markdown Support**: Write your posts in Markdown
- **React Components**: Embed React components in your posts
- **Syntax Highlighting**: Code blocks are automatically highlighted
- **SEO Optimized**: Each post has proper meta tags

## Code Example

Here's a code example with syntax highlighting:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```
